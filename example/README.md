# Projeto de Exemplo para Agente QA

Esta pasta contem um projeto simples em Node.js usado como ambiente de exemplo para testes do agente QA.

## Objetivo

Permitir que o agente QA valide funcionalidades basicas de API, incluindo:

- CRUD de usuarios
- Validacao de campos obrigatorios
- Validacao de formato de email

## Estrutura

- `index.js`: API Express com dados mockados em memoria
- `package.json`: configuracao do projeto e dependencias

## Como executar

1. Instale as dependencias:

```bash
npm install
```

2. Inicie o servidor:

```bash
npm start
```

3. A API sera iniciada em:

`http://localhost:3000`

---

Este projeto e destinado apenas para testes e demonstracoes.
